You can load content into the background while all windows stay visible. This is a little trick, because the background is a 100% window (.Bg=true) with an empty skin. So it looks like the real background page. To use this feature, every background page must have the following event: onLoad="parent.WinLIKE.resizewindows();".

When you want to change background dynamicaly without reloading the window, you must also actualize the background window size with WinLIKE.resizewindows(). Click the bars below to test it.
